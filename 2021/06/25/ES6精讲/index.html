2021-06-25 ES6-7 ES6 评论

箭头函数

简写函数，需要注意两点：

  1. 如果有且仅有一个参数，()也可以不写

  2. 如果有且仅有一个语句并且是return，{return }也可以不写

this

展开符（三个点…）

相当于arr1的三个元素全都作为参数a了，并且函数体中的+号就变成了字符串拼接了，arr1的三个参数就用,号拼接了。

Array方法

map 映射：一一对应（进去几个，出来对应的几个）

reduce 缩减（进去n个，出来就剩1个）

常见场景：求平均分，求和

第一次循环tmp是第一个元素68，item是它后面的那个即第二个元素53，index指的第几次运算（也可以理解为当前item对应的脚标）。

filter 过滤 （进去n个，出来几个不一定，得符合条件）

forEach 遍历，没有返回值

模块

  • 常见的导出方式（声明模块）：

    1. 声明并导出

      export let a = 12

      export const a = 12

    2. 导出

      export {name1, name2, ...}

    3. 导出方法

      export function show() { ... }

    4. 导出类，react和vue的组件都用的这种方式

      export class ClassName { ... }

    5. 导出默认成员

      export default xx

  • 常见的引入方式：

    1. import * as mod1 from 'xxx' 引入所有export的成员
    2. import mod1 from 'xxx' 引入default成员
    3. import {a, b as name2, [...]} from 'xxx' 选择引入成员
    4. import 'xxx' 只引入，常用于引入css
    5. let promise = import('./mod1') 异步引入，import引入的结果是个promise，可以用来做路由懒加载
